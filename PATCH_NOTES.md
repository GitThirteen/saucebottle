This update introduces multiple hotfixes.

### Changelog

#### Improvements
* LMB and RMB now have different behaviors when clicking on the folder path box: LMB opens the folder in your system's explorer, RMB copies the absolute path.

#### Bugfixes
* Fixed a bug that resulted original work with a registered fandom to display `.original` as the fandom.
* Copying the path in the path box will now yield the correct folder path.
* Developing in Vite will now no longer stack drag-n-drop listeners on hot reload.
* Fixed the run button in the header and the auto listen/process folder setting exhibit the exact same behavior.

### Installation

1. Download the appropriate installer for your operating system from the **Assets** section below (`.exe` for Windows, `.dmg` or `.app` for macOS, `.deb` or `AppImage` for Linux).
2. Run the installer and launch SauceBottle.
